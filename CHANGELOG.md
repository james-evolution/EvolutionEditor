
# Changelog

All notable changes to this project are documented here.

This project follows [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) and [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.0.0] - 2026-02-16

### Added
- Modular, block-based rich text editor for React
- Paragraph blocks with text formatting
- Heading blocks (H1, H2, H3)
- Ordered and unordered lists (with nesting)
- Image embeds via URL (with alt text and custom styles)
- YouTube video embeds (multiple URL formats supported)
- Horizontal divider blocks
- Text color picker
- Font size selector
- JSON import/export (clean, portable structure)
- Read-only and editable modes
- Placeholder support for empty editors
- Toolbar with formatting and block controls
- Customizable styles via CSS variables and config
- Responsive, accessible design
- Example application and usage guide
- Rollup build configuration for npm publishing
- React 16.8+ support (React 18+ recommended)

### Developer Features
- Extensible block/component architecture
- Easy to add new block types
- Utility helpers for serialization/deserialization
- Well-documented codebase and guides
- Comprehensive styling system (CSS modules and variables)
